# "ZELLIT" — App Branding

**Date**: 2026-06-04  
**Context**: About screen needs its own identity. User proposed "ZELLIT" — a portmanteau.

## The Name

**ZELLIT** = ZELLIj + Themes (drop "heme" + "j" → ZELLIT)

Pronounced like "zel-it" — short, punchy, two syllables.

What it signals: "Zellij it" — as in, "theme it", "style it", "make it yours". Works as both a noun (this tool is ZELLIT) and a verb (zellit your terminal).

Honest assessment: **it's good**. Three reasons:

1. **Clear lineage** — immediately tells Zellij users what ecosystem this is for, unlike "tab-config" which sounds like a debug tool
2. **Memorable** — short enough to type, distinct enough to search for
3. **Implies action** — "zellit" reads like "let's style this thing", which is exactly what the tool does
